# ทำไมเราถึงใช้ Bootstrap

- Backward Compatible for old browser
- Ready-Mad UI Component
- Ready for Layout easier e.g. Flexbox
- Responsive Design
- JS included, no need to implement JS for interactive UI
- Rapid Development
- Huge Community
