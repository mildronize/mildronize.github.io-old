# บันทึก

ถ้าเป็น static file เช่น

```
https://thadaw.com/post/index.html == https://thadaw.com/post/
```

แต่ถ้าใช้

```
https://thadaw.com/post มันจะ redirect ไปยัง https://thadaw.com/post/
```

ถ้าใน Facebook debugger มันจะโชว์
